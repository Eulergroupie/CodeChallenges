CountTheDivisorsOfANumber.js

Count the number of divisors of a positive integer n.

    function getDivisors(n) {
        let r = [];
        for(let i = 1; i <= (n/2); i++) 
          if(n % i == 0) 
            r.push(i);
        r.push(n);
        return r;
    }
      
    function getDivisorsCnt(n) {
        let r = 0;
        for(let i = 1; i <= (n/2); i++) 
          if(n % i == 0) 
            r++;
        return ++r;
    }

Listing the divisors and counting the divisors use the same process of dividing by every positive integer and checking for remainders.

DescendingOrder.js

Given a non-negative integer, return the same number with its digits in descending order. 

    const descendingOrder = n => Number([...`${n}`].sort().reverse().join(''));

Convert the number to a string and then an array of characters. Sort in ascending order, then reverse it and convert it back into a string and finally a number.
