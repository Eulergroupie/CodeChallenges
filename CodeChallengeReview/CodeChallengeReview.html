ASCIITotal.js from CodeWars 8kyu

Given a string return the sum of all character ASCII values as an integer.

function uniTotal(s) { 
    if(typeof(s) !== 'string')
        return null;
                  
    return [...s].reduce((t, c) => t + c.charCodeAt(0), 0);
}

It is good practice to check inputs to a function with type checks. The last line splits the string into a character array and uses the iterator function reduce to add up the character codes.

FindTheFirstNonConsecutiveNumber.js from CodeWars 8kyu

Find the first element of an array that is a non-consecutive integer, otherwise return null. 

function firstNonConsecutive (arr) {
    const len = arr ? arr.length : 0;  // assign length based on the existence of the given array
    if(len > 1)                      // if there are 2 or more elements...
      for(let i = 1; i < len; i++)  // loop thru input array starting with 2nd element
        if(arr[i] !== arr[i - 1] + 1)  // ensure the previous element was one less
          return arr[i];  // non-consecutive!
    
    return null;  // didn't find anything non-consecutive
}

The first line handles undefined inputs and empty arrays. The second line handles single element arrays. A simple for loop stepping through the array while looking back a step works well here because you can easily start on the second element (avoiding an index error by looking back on the first element).

ReturnTwoHighestValuesInList.js from CodeWars 8kyu

Find the two distinct highest values in a list. Return as a list from highest to lowest

const twoHighest = r => [...new Set(r.sort((a,b) => b - a))].slice(0,2);

Sort the list in descending order, and place it in a set to remove duplicates. Then turn it back into a list containing only the first two values.

SumOfPositive.js from CodeWars 8kyu

Find each positive value in a list and return the sum.

// C-style
function positiveSum(arr) {
    let sum = 0;
    for(let i = 0; i < arr.length; ++i) {
        if(arr[i] > 0)
            sum += arr[i];
    }
    return sum;
}
    
// forEach with anonymous function
function positiveSum(arr) {
    let sum = 0;
    arr.forEach(value => value > 0 ? sum += value : 0);
    return sum;
}
    
// reduce lives up to its name
const positiveSum = arr => arr.reduce((sum, x) => sum + (x > 0 ? x : 0), 0);

This evolved from a C style of coding to JavaScript. The for loop that keeps a running total checked by an if statement is a standard application of loops. Switching to an iterator function that mimicks the for loop and a conditional statement instead of an if, seem like Javascript, but it's really a one-for-one replacement. Using the overall form of a function as an object in the shape of an arrow function while also using the iterator specifically made for running sums makes the full transition to Javascript style.
